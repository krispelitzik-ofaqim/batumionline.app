# CLAUDE.md — Batumionline.app

## מה הפרויקט

אפליקציה מובייל בעברית (RTL מלא) לתיירים ישראלים בבטומי, גאורגיה.
מיועדת לחנויות Apple App Store ו-Google Play Store.
90% שימוש על נייד, 10% iPad.

---

## סטאק טכני

- **Frontend:** React Native + Expo
- **שפה:** TypeScript
- **ניווט:** React Navigation
- **Backend:** Node.js + Express (פרויקט נפרד, על Render)
- **מפות:** Google Maps API (`react-native-maps`)
- **אודיו:** MP3 (`expo-av`)
- **RTL:** `I18nManager.forceRTL(true)` מההתחלה
- **גיבוי:** GitHub

## עיצוב

- Mobile First — נייד קודם, iPad משני
- עיצוב קליל, מודרני, קומפקטי
- קארדים עם תמונה + כותרת + תיאור קצר
- רספונסיבי: נייד = 2 קארדים בשורה, iPad = 3 קארדים בשורה
- `useWindowDimensions()` בכל קומפוננטה שצריך
- גופן: System Font (San Francisco / Roboto)

## כללי עבודה

- **RTL מהיום הראשון** — כל layout ו-flexDirection מותאם לעברית
- **Mobile First** — תמיד לבדוק על 375px (iPhone SE) לפני iPad
- **אין hardcoded strings** — כל טקסט במשתנה לקראת תרגום עתידי
- **תמונות מהדרייב** — תמונות מורשות לשימוש, נטען מ-assets/images
- **MP3** — קבצי אודיו ב-assets/audio, נגן עם expo-av
- **כל push = commit משמעותי** — לא לדחוף קוד שבור
- **Backend נפרד לחלוטין** — לא לערבב קוד האפליקציה עם הניהול
